# VideoTrend - 智能视频趋势与流量分析系统

## 项目简介
VideoTrend 是一款专为短视频创作者、MCN 机构和品牌主设计的全域社交媒体分析与模拟投流系统。通过接入 TikHub 的实时视频抓取能力、Bocha Web Search API 的全网趋势检索能力，以及大模型的深度内容理解/流量预测引擎，VideoTrend 能够结合当下最热门的社会事件与用户心智，帮助用户在正式投放前精准预判视频的传播潜力与破圈可能。

## 用户故事
- **创作者张三**：希望在视频发布前，不仅根据视频本身，还要结合当下最火的网络热点，由 AI 来模拟不同平台的潜在传播效果。
- **MCN 负责人李四**：需要快速审计旗下博主的商业价值，并结合最新的社会消费趋势，为品牌方提供结构化的数据支持和受众洞察。
- **品牌经理王五**：在进行矩阵投流时，希望有一个“沙盘”能预测预算分配的最优方案，并基于当前行业的最新动向来降低实测成本。

## 解决的问题
1. **多平台碎片化数据获取难**：一站式抓取并展示抖音、B站、小红书等主流平台的账号与视频数据。同时支持将跨平台的同属于一个 IP 的账号组合为“矩阵”，集中进行粉丝和数据汇集。
2. **账号异动监控滞后**：提供“每日自动监控”功能，锁定每天早9点追踪最新作品表现并对比历史均值，实时通过飞书机器人进行信息预警。
3. **商业价值评估与环境脱节**：系统通过检索实时的互联网数据与讨论焦点进行背景对齐，结合大模型有效且客观地审计博主的原创力与品牌适配度。
4. **投流策略依靠盲测**：通过数据沙盘模拟技术与外部素材的自定义分析（支持手动上传洞察报告与视频文件），全副武装地降低跨平台投放的盲目性。

## 产品价值
- **决策科学化**：依靠结构化数据与网络热点的深度对焦模型，从经验决策转向数据验证。
- **效率极致化**：分钟级全自动归组矩阵、监控博主互动数据并生成带有实时讨论基础的市场洞察报告。
- **成本可控化**：支持用户预先载入特定分析文档，在真实资金投入前进行多维度的虚拟演练，显著提升投放 ROI。

## SPI (Situation-Pain-Impact/Insight)
- **S (Situation)**: 内容热点稍纵即逝，且各平台的算法与受众讨论焦点每天都在变。
- **P (Pain)**: 单一依赖视频数据进行分析，会忽略“天时地利”等大环境因素；同时缺乏跨平台的直观整合能力，使得投流预算分配充满不确定性。
- **I (Impact)**: 采用“矩阵归位”管理，并主动联接全网实时搜索引擎对齐背景知识，实现了投流预估在“时空”维度的精准提取，投流 ROI 数据准确性得到有效保证。
